# Regras de Formatação do Wiki Quero-Quero
1. Toda nota deve começar com tags (ex: #dev-container, #reuniao).
2. Se houver tarefas no texto, extraia-as para uma seção final chamada "## Pendências" usando checkboxes `- [ ]`.
3. Use bullet points para facilitar a leitura rápida.
4. Mantenha um tom profissional.
